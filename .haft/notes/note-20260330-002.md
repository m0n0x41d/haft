# Represent pasted images as typed user message parts

## Rationale

Inline <file> pseudo-tags are acceptable for text attachments but wrong for images. A dedicated image part keeps multimodal payload construction at the provider boundary and avoids smuggling binary content through plain text.

- `internal/agent/types.go`
